# T1218.011 — Rundll32 Abuse

## What This Attack Does

The attacker uses rundll32.exe — a legitimate Windows tool — to execute
malicious code. Because rundll32 is a trusted Windows binary, many
security tools and antivirus programs ignore it or trust it by default.
This technique is called Living off the Land — using Windows own tools
against itself so the attacker blends in with normal system activity.

## How I Simulated It

Tool: Atomic Red Team
Command executed on Windows Server target:

Invoke-AtomicTest T1218.011 -TestNumbers 1

## What Splunk Captured

Sysmon Event ID 1 (Process Creation) fired showing:
- Image: C:\Windows\System32\rundll32.exe
- User: WINDOWS-SERVER\Administrator
- CommandLine: rundll32.exe javascript:"\..\mshtml,RunHTMLApplication"

## SPL Detection Query

index=* source="WinEventLog:Microsoft-Windows-Sysmon/Operational"
EventCode=1 Image="*rundll32.exe" CommandLine="*javascript*" OR CommandLine="*http*"
| table _time, ComputerName, User, CommandLine, ParentImage

## Result

Splunk successfully detected suspicious rundll32 execution.
Alert fired within seconds of the command running on the target.
